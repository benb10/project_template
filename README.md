# project_template

## Frontend
- react app 
- created with `pnpm create next-app -e https://github.com/jpedroschmitz/typescript-nextjs-starter`
- deployed using vercel https://vercel.com

| Env   | URL                                      |
|-------|------------------------------------------|
| local | http://localhost:3000                    |
| prod  | https://project-template-beta.vercel.app |

### local dev
```bash
$ ./fe.sh
```
